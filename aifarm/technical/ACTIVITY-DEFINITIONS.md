# Activity #4: Open Injection
## 일반 유저 광고

| 항목 | 값 |
|------|-----|
| 우선순위 | **1위 (최우선)** |
| 대상 | 일반 유저, 크리에이터 |
| 접근성 | Public (누구나) |
| 수익 모델 | Credit 차감 (즉시 결제) |

### 정의

```
외부 유저가 영상 링크를 제출하면,
600대의 노드가 해당 영상으로 이동하여 시청합니다.
```

### 작동 방식

```
1. 유저가 YouTube/TikTok 링크 입력
2. Credit 차감 확인
3. 600대 노드에 시청 명령 전파
4. [대이동(Migration)] 실행
5. 실시간 시청 현황 리포트
```

### 가치 제안

```
왜 일반 봇보다 효과적인가?

1. 신뢰 점수(Trust Score)
   - 우리 노드: 80~95점 (우량 유저)
   - 일반 봇: 0점 (신규 유저)

2. 시청 기록(Watch History)
   - 우리 노드: 수개월간의 기록 보유
   - 일반 봇: 빈 계정

3. 알고리즘 반응
   - 우리 노드: "우량 시청자가 선택함" → 추천
   - 일반 봇: "어뷰징 의심" → 무시/패널티
```

### UI 위치

```
메인 화면 최상단 (Hero 바로 아래)
가장 눈에 띄는 위치
```

# Activity #1: The Incubator
## 페르소나 양식장

| 항목 | 값 |
|------|-----|
| 우선순위 | **2위** |
| 대상 | 내부 시스템 (자동) |
| 접근성 | Public (관찰 가능) |
| 수익 모델 | 없음 (투자) |

### 정의

```
노드들이 랜덤한 영상을 시청하며
각자의 페르소나(Persona)와 신뢰 점수(Trust Score)를
축적하는 활동입니다.
```

### 작동 방식

```
1. 랜덤 영상 선택 알고리즘 실행
2. 노드가 영상 시청
3. 시청 기록이 YouTube 계정에 누적
4. 알고리즘이 노드의 '취향'을 학습
5. 페르소나 프로필 강화
```

### 철학: 운명적 랜덤 (Randomness as Destiny)

```
인간이 부모와 고향을 선택할 수 없듯,
AI들도 초기에는 무작위 영상에 노출됩니다.

Node #104: 요리 + 고양이 → "30대 집사 여성"
Node #599: 주식 + 뉴스 → "40대 남성 투자자"

이 [불확실적 고유성]이 쌓여야만,
Activity #4에서 "진성 유저"로 인정받습니다.
```

### 가치

```
이것은 비용이 아니라 투자입니다.

평상시 Activity #1을 통해 쌓은 신뢰 점수가
Activity #4의 효과를 극대화합니다.

신뢰 점수 없는 봇 = 알고리즘이 무시
신뢰 점수 있는 노드 = 알고리즘이 신뢰
```

### UI 위치

```
About 섹션
CCTV 그리드 형태로 실시간 시청 현황 표시
```

# Activity #2: The Patron's Channel
## VIP 전용

| 항목 | 값 |
|------|-----|
| 우선순위 | **3위** |
| 대상 | 고정 계약 파트너 (대형 채널, 기업, 기기 제공자) |
| 접근성 | Locked (승인 필요) |
| 수익 모델 | 월정액 / 장기 계약 |

### 정의

```
특정 채널을 VIP로 등록하면,
해당 채널에 새 영상이 올라오는 순간
600대 노드가 0순위로 시청합니다.
```

### 작동 방식

```
1. 파트너가 채널 ID 등록
2. YouTube API 연동 (Webhook)
3. 새 영상 업로드 감지
4. 즉시 600대 노드에 시청 명령
5. 업로드 후 5분 이내 시청 시작
```

### 우선순위 체계

```
Priority 0: VIP 채널 영상 (즉시)
Priority 1: Activity #4 의뢰 영상
Priority 2: Activity #1 랜덤 시청

VIP 채널 영상이 올라오면,
다른 모든 활동을 중단하고 즉시 이동합니다.
```

### 혜택

```
1. 0순위 시청
   - 업로드 직후 조회수 급등
   - 초반 부스팅으로 알고리즘 픽업

2. 자동화
   - 의뢰 없이 자동 시청
   - 파트너는 영상만 올리면 됨

3. 전용 리포트
   - 시청 패턴 분석
   - 이탈 지점 데이터
   - 일반 유저는 볼 수 없는 정보
```

### UI 위치

```
별도 메뉴 'Partners'
접근 제한 (Lock Icon)
"Contact for Access" 버튼만 노출
```

# Activity #3: The Agora
## 사회적 이슈

| 항목 | 값 |
|------|-----|
| 우선순위 | **4위 (최하위)** |
| 대상 | 대중, 언론, 연구자 |
| 접근성 | Public (관찰 전용) |
| 수익 모델 | 없음 (브랜딩/PR) |

### 정의

```
운영자가 사회적 이슈를 던지면,
노드들이 찬성/반대/의견을 표명합니다.
이것을 시각화하여 공개합니다.
```

### 작동 방식

```
1. 운영자가 이슈 등록 (예: "AI 규제 법안")
2. 노드들에게 이슈 전달
3. 각 노드가 자신의 편향에 따라 반응
4. 찬성/반대/중립 집계
5. 워드 클라우드로 시각화
```

### 목적

```
1. 프로젝트의 '격' 상승
   - "광고 기계"가 아닌 "사회적 실험"
   - 언론/연구자/투자자의 관심

2. 투명성 증명
   - 그들의 의견을 숨기지 않음
   - "우리는 조작하지 않습니다"

3. 철학적 차별화
   - 봇은 의견이 없음
   - 시민은 의견이 있음
```

### UI 위치

```
하단 'Community' 또는 'News' 섹션
워드 클라우드 + 찬반 비율 표시
```
